
# E-Commerce Frontend

Este é o front-end de um sistema de E-Commerce, que permite aos usuários cadastrar e visualizar produtos, além de entrar em contato com vendedores através de um chat.

## Tecnologias Utilizadas

- **React**: Biblioteca JavaScript para construção de interfaces de usuário.
- **Material-UI**: Framework de design para React que proporciona componentes prontos para uso.
- **Axios**: Biblioteca para realizar chamadas HTTP para o backend.
- **React Router**: Para gerenciamento de rotas na aplicação.
- **Socket.IO**: Para comunicação em tempo real via WebSocket.

## Funcionalidades

- **Cadastro de Produtos**: Os usuários podem cadastrar novos produtos para venda.
- **Listagem de Produtos**: A aplicação exibe uma lista de produtos disponíveis para compra.
- **Detalhes do Produto**: Cada produto possui uma página dedicada com detalhes e informações do vendedor.
- **Chat com Vendedores**: Os usuários podem se comunicar com os vendedores por meio de um chat integrado.

## Como Executar a Aplicação

1. Clone o repositório.
2. Navegue até o diretório do front-end.
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie a aplicação:
   ```bash
   npm start
   ```

A aplicação estará disponível em `http://localhost:3000`.
